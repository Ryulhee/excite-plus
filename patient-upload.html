1. 연구 태그 선택

연구 단위 선택

태그는 연구를 구분하는 큰 라벨입니다. Day, 대조군, 처치군 같은 값은 선택한 연구 태그 안에서 각 환자/파일/시점에 입력되는 연구 내 분류값입니다.

1-1. 연구 대상자 배정

이미 등록된 환자 master에서 이 연구 태그에 포함할 대상자를 선택합니다. 환자 정보는 고정이고, 비교군·기준시점·관찰구간·검사항목은 아래 단계에서 연구별로 반복됩니다.

선택가명 ID등록번호성별출생연도상태
배정할 환자 master가 없습니다.
먼저 고정 환자 정보를 등록하세요.

2. 연구 내 분류값 선택

연구 내 분류값 직접 작성필요할 때만 입력
직접 작성한 값은 저장 시 표준 드롭다운 후보로 등록할 수 있게 합니다. 예: 2days, day2, D2처럼 흩어지지 않도록 대표 표기와 동의어를 함께 관리합니다.
예: ECMO군은 Pre-index/During ECMO/Post-index 궤적으로 여러 자료가 반복될 수 있고, Non-ECMO군은 ICU admission 또는 pseudo-index time을 기준으로 적은 관측값이나 단일 관측값을 연결할 수 있습니다.

3. 입력할 자료 항목 직접 고르기

자료 항목 직접 작성custom variable

4. 새 변수 만들기

변수 생성은 일부러 단순하게 유지합니다. 변수명, 자료형, 단위만 입력하고 저장하면 변수 사전에 계속 누적됩니다. 연구 내 분류값, 환자 매칭, Day/Group 정보는 이후 단계에서 따로 붙입니다.

예: lactate 저장 → CRP 저장 → culture_result 저장처럼 계속 추가합니다. 각 변수를 어느 환자, 어느 Day, 어느 그룹에 붙일지는 아래 입력 예정 자료에서 따로 연결합니다.
